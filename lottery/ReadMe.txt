This folder contains the code for the Smart Contract creation, as well as the tests for
its functionality. 

To perform the validation, follow these steps: 
1. Open Powershell on lottery folder. 
2. Run npm install
2.1 Run npm install xhr2
3. Open the Lottery.test.js file in the terminal and run npm run test
4. If you want to deploy the contract again, open the deploy.js file in terminal
5. Run npm install @truffle/hdwallet-provider
6. Run node deploy.js 

To check the already deployed contract for the Hackathon, paste the contract address: 
0x9B2547E015De1266613E463429f9DCbA54269806
in the Etherscan Rinkeby platform (https://rinkeby.etherscan.io/)
